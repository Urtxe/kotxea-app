# 🚗 Calculadora de Gastos de Viaje (Kotxea)

¡Bienvenido! Esta es una herramienta sencilla creada con HTML, CSS y JavaScript para calcular y dividir los gastos de los viajes en coche entre amigos. 

Registra quién conduce y quiénes viajan para mantener un balance justo.

## 🔧 Cómo funciona

* **Gidaria:** Selecciona quién condujo el coche.
* **Nortzuk joan dira?:** Marca las casillas de todas las personas que iban en el viaje (incluido el conductor).
* **Bidai berria gehitu:** El botón recalcula todo.

La tabla muestra:
* **Bidaiak:** El "coste" o parte de los viajes que te corresponde pagar.
* **Gidatu:** Las veces que has conducido (lo que "te deben").
* **Emaitza:** El balance. Un número positivo significa que debes dinero (has viajado más de lo que has conducido), y uno negativo significa que te deben.
